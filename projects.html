Some Stuff I've Worked On

  • ML/Sports analytics blog

    A personal blog built with React, Gatsby.js and GraphQL. Each commit triggers an auto-deployment so the content is always up-to-date. In this blog I share results of me collecting, cleaning and analyzing sports data as well as some projects / notes of me experimenting with image recognition and NLP.

  • NBA Chrome Extension

    A chrome extension that is made for fans to be able to easily keep track of the scores of their favorite team and their scores from previous games. Published on chrome store with 5 star rating.

  • Conway's Game of Life

    A 'zero-player-game' built with React that imitates cellular automaton. The player will be able to change generation speed, grid size as well as generate pre-built patterns. Live official site is deployed on surge.sh.

  • Tweet Generator (WIP)

    A machine learning project that aims to be able to learn and generate new tweets based on a user's tweet history. Using LSTM models and RNNs to train the model and use adam and gradient descent in back propagation to reinforce the training. It's been training on a google colab vitrual machine with GPUs. The jupyter notebook is available on my github.
